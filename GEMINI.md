# GEMINI.md - Project Overview & Instructions

## Project Overview

This repository, **college-app-programming-apply** (앱 프로그래밍 응용), serves as a workspace for 3rd-year college course assignments and personal web development projects. It primarily focuses on front-end development, personal branding, and exploring AI-assisted coding methodologies (referred to as "Vibe Coding").

### Core Technologies

- **HTML5 / CSS3**: Used for structuring and styling the portfolio and study applications.
- **JavaScript (Vanilla)**: Powers interactivity in applications like TodoFlow.
- **AI-Assisted Development**: Integration of AI agent instructions (`AGENTS.md`, `CLAUDE.md`) to streamline the coding process.

## Usage & Development

### Running the Projects

The current projects are static web applications. To view them:

- Open `Portfolio/portfolio.html` in any modern web browser.
- Open `VibeCoding/Study-02/index.html` to use the TodoFlow application.

### Development Conventions

1. **Static First**: Prefer vanilla HTML/CSS/JS for assignments to ensure portability and ease of review.
2. **AI Collaboration**: Follow the guidelines established in `VibeCoding/Study-01/AGENTS.md` when using AI agents for code generation.
3. **Style**: Use CSS Custom Properties (Variables) for consistent theming and Flexbox/Grid for modern layouts.
