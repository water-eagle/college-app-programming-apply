// Philosophy

개발 철학

🔒
Privacy by Design
기능 추가보다 사용자 데이터 보호를 먼저 설계합니다. 수집 최소화, 암호화, 접근 제어를 기본값으로 합니다.
⚡
성능 = 신뢰
느린 서비스는 사용자를 잃게 합니다. Core Web Vitals를 매 배포마다 측정하고 개선합니다.
📚
지속적 학습
매주 새로운 기술 트렌드를 팔로우하고, 학습한 내용을 실제 프로젝트에 빠르게 적용합니다.
// 개인정보보호 선언
코드 한 줄이 누군가의 일상을 지킵니다.
  • 개인정보는 목적에 필요한 최소한만 수집합니다
  • 수집한 데이터는 AES-256 이상으로 암호화합니다
  • 사용자가 요청하면 즉시 데이터를 삭제합니다
  • 제3자 공유 시 반드시 명시적 동의를 받습니다
  • 개인정보 관련 법령 변경을 정기적으로 확인합니다
